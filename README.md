# 🦅 Eagle_Vision_AI

An AI-powered classroom monitoring system developed using **Python**, **OpenCV**, **InsightFace**, **MediaPipe**, and **SQLite**.

The project performs **real-time face recognition**, **automatic attendance marking**, and **student attention monitoring** using a webcam.

---

# 📌 Project Overview

Eagle_Vision_AI is designed to automate classroom monitoring by recognizing students, marking attendance, and analyzing attention levels through eye monitoring.

The project follows a modular architecture to ensure clean, maintainable, and scalable code.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Eagle_Vision_AI.git
```

Move into the project

```bash
cd Eagle_Vision_AI
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

```bash
python app.py
```
